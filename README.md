Sistema de Alarme simples em C com placa Raspberry Pi Pico

Descrição do harware

Raspberry Pi Pico: Configuração padrão com pinos configurados para leitura dos botões e controle do buzzer.
Botões: Configurados como entradas digitais com pull-up ou pull-down.
Buzzer: Configurado para emitir som ao receber sinal de saída.

Metodologia
A metodologia seguida envolveu a pesquisa de sistemas simples de alarme e a escolha do hardware adequado. 
O software foi desenvolvido com base em funções básicas de controle de GPIO, temporização e interação com botões.
